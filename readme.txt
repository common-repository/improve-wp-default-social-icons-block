Enhance WP WordPress Default Social Icons block by extending it. It provides rounded, square social icons styles. You can give icon color and background color to social icons.

== Description ==

Enhance WP WordPress Default Social Icons block plugin extends WordPress default Social Icons block. It adds Icon Background color, Icon color, Hover Background color and Hover Icon color.

It gives Rounded Shape, Square Shape, Diamond Square Shape, Diamond Rounded Square Shape, Parallelogram Shape and Parallelogram Rounded Shape styles.

Features:

*   Match Social Icons colors to your theme's color scheme.
*   Set hover colors.
*   Different shape of Social Icons.

== Installation ==


1. Upload `improve-wp-default-social-icons-block.zip` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Add or Edit Post and Add Social Icons block.

== Frequently Asked Questions ==

= How to select Social Icons blocks after adding a Social Icon? =

Please click on the horizontal space where icon doesn't exist and the Social Icons block will be selected.
